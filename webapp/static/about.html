About the Project

Agnirodhak is a research-focused early warning system that estimates short-term forest fire risk for Uttarakhand using daily remote sensing and reanalysis data.

The project combines stacked GeoTIFF inputs (temperature, wind, NDVI, land cover, terrain) with machine learning models to produce next-day burn probability maps at the pixel level.

  • Study region: Uttarakhand, Himalayan foothills.
  • Data sources: ERA5 (meteorology), MODIS (NDVI, burned area), ESA WorldCover (LULC), SRTM (DEM / terrain).
  • Models explored: Random Forest baseline, multiple U-Net variants, and ConvLSTM temporal models.
  • Operational model: U-Net Scenario 1 (predicting next-day burn risk from a single-day stack).

This web interface is meant for visualization and outreach. It should not be used as the sole basis for operational firefighting decisions without additional validation and expert review.
